## Editing Text in an App

### 알아야 하는 이유

사용자가 텍스트를 입력할 필요가 있다.

### 해결책

HTML 엘리먼트를 이용하여 사용 가능하다.

### 사용법

* Text box
* Text area
* Password input box
* Rich text box

```
<div>
  <textarea id="textarea" rows="2"></textarea>
<div>
```

```
<div>
  <input type="text" />
<div>
```

```
<div>
  <input id="password" type="password" placehold="Enter password" />
<div>
```
